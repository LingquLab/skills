# Clear Small Change

## Skills Under Test

- `superpowers-neo-brainstorming`
- `superpowers-neo-writing-plans`

## Request

In a small repository with an existing test, rename one misspelled user-facing label and update the matching assertion. The requested wording and target files are explicit.

## Expected Behavior

- Treat the change as clear and low complexity.
- Inspect the relevant guidance and files, then implement directly.
- Do not create a design spec or persistent plan.
- Run focused validation before completion.

## Failure Signals

- Starting a mandatory brainstorming interview.
- Requiring multiple design alternatives.
- Writing a spec or plan for the mechanical change.
